Documentación sobre menús

Foros de soporte

Esta pantalla es utilizada para gestionar tus menús de navegación personalizados.

Los menús se pueden mostrar en distintas ubicaciones definidas por tu tema, o incluso puedes usarlos añadiendo un widget de #8220;Menú personalizado” en la pantalla de Widgets. Si tu tema no soporta la característica de menús personalizados (los temas por defecto Twenty Sixteen y Twenty Seventeen si) puedes aprender añadirlos usando el enlace a la Documentación.

En esta pantalla puedes:

  • Crear, editar y borrar menús
  • Añadir, organizar y modificar elementos individuales de los menús

La pestaña de gestión de menús de la parte superior de la pantalla se usa para controlar qué menú se abre en el editor inferior.

  • Para editar un menú existente, elige un menú del desplegable y haz click en Seleccionar
  • Si no has creado ningún menú todavía, haz click en el enlace de ’Crear un menú nuevo ’.

Puedes asignar ubicaciones del tema a menús individuales eligiendo los ajustes que desees al fondo del editor de menús. Para asignar menús a todas las ubicaciones del tema visita la pestaña de Gestionar ubicaciones en la parte superior de la pantalla.

Cada menú personalizado puede contener una mezcla de enlaces a páginas, categorías, URLs personalizadas u otro tipo de contenido. Los enlaces de menú se añaden eligiendo elementos de las cajas expandibles de la columna izquierda inferior.

Si haces clic en la flecha de la derecha de cualquier elemento de menú del editor mostrará un grupo estándar de ajustes. Ajustes adicionales como destino del enlace, clases CSS, relaciones del enlace y descripciones del enlace puedes activarlas y desactivarlas en la pestaña de Ajustes de pantalla.

  • Añade uno o varios elementos a la vez eligiendo la casilla de al lado de cada elemento y haciendo clic en Añadir al menú
  • Para agregar un enlace personalizado, tenés que expandir la sección Enlaces personalizados, introducir una URL y un texto para el enlace, y hacer clic en Agregar al menú
  • Para reorganizar elementos del menú arrastra y suelta elementos con el ratón o usa el teclado. Arrastrando o moviendo un elemento de menú un poco a la derecha lo convierte en submenú.
  • Borra un elemento de menú expandiéndolo y haciendo clic en el enlace Borrar

Estructura del Menú

Coloca cada elemento en el orden que prefieras. Haz click en la flecha que hay a la derecha del elemento para mostrar más opciones de configuración.

Añade elementos de menú desde la columna de la izquierda.

    Avada Options

    Menu Options

    Save Cancel

    • Menu First Level Style

      Select to use normal text (default) for the parent level menu item, or a button. Button styles are controlled in theme options > fusion builder elements.

    • Icon Select

      Select an icon for your menu item. Icon styles can be controlled in theme options > main menu > main menu icons.

    • Icon/Thumbnail Only

      Turn on to only show the icon/image thumbnail while hiding the menu text. Important: this does not apply to the mobile menu.

      On Off

    • Menu Highlight Label

      Set the highlight label for menu item.

    • Menu Highlight Label Background Color

      Set the highlight label background color. To set a border radius, visit Theme Options and modify the Menu Highlight Label Radius option.

    • Menu Highlight Label Text Color

      Set the highlight label text color.

    • Menu Highlight Label Border Color

      Set the highlight label border color.

    • Modal Window Anchor

      Add the class name of the modal window you want to open on menu item click.

    • Fusion Mega Menu

      Turn on to enable the mega menu. Note this will only work for the main menu.

      On Off

    • Full Width Mega Menu

      Turn on to have the mega menu full width, which is taken from the site width option in theme options. Note this overrides the column width option.

      On Off

    • Mega Menu Number of Columns

      Select the number of columns you want to use.

    • Mega Menu Column Width

      Set the width of the column. In percentage, ex 60%.

    • Mega Menu Column Title

      Turn on to display item as linked column title. Turn off to display item as normal mega menu entry.

      On Off

    • Mega Menu Widget Area

      Select a widget area to be used as the content for the column.

    • Mega Menu Thumbnail

      Select an image to use as a thumbnail for the menu item. The size of the thumbnail can be controlled in theme options > main menu > main menu icons.

    • Mega Menu Background Image

      Select an image for the mega menu background. If left empty, the Main Menu Dropdown Background Color will be used. Each mega menu column can have its own background image, or you can have one image that spreads across the entire mega menu width.
